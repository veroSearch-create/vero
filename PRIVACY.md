# Privacy Policy

Search Sanity collects no data and sends no data anywhere.

- **No analytics.** Zero usage tracking of any kind.
- **No external requests.** The extension makes zero network requests to non-Google domains during normal SERP browsing.
- **Local storage only.** Your settings are stored in `chrome.storage.sync`, which Chrome may sync to your Google account if you have Chrome Sync enabled — this is standard Chrome behavior and is not controlled by this extension.
- **Open source.** Every line of code is readable in the `src/` directory.

The extension reads the Google Search results page DOM to apply visual treatments. It does not read, store, transmit, or log the content of your searches or browsing activity.
